# FlyUI 苍蝇UI

## 介绍

如你所见, 这是一个 UI 库, 或者严格意义上来讲, 这是一个 Windowless GUI 框架. 由 Fly 以及他的好基友 EnderMo, 咸鱼好友 Null 参与开发 (参与了个寂寞了属于是)

FlyUI 从 0 开始写起, 他将具有强大的性能, 并且具备 Windows 原生窗口所不具备的复杂效果以及特性. 它完美兼容前端页面的各种布局方式, 你也可以直接选择使用 Windows 原生布局方式或者 WPF 布局方式, 通过继承接口, 用户可以实现更多绘图系统, 以实现跨平台开发.

目前 FlyUI 仅支持 Windows 操作系统, 并且基于 WinAPI 开发.
